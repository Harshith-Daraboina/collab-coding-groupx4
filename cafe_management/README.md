# Cafe Management System

A modular Python CLI application to manage cafe operations.

## Features
- **Menu Management**: View available items and prices.
- **Order Management**: Add items to a customer's order.
- **Billing**: Calculate totals and generate bills.

## Project Structure
- `menu.py`: Manages menu items.
- `order.py`: Handles order creation and tracking.
- `billing.py`: Calculates bills.
- `display.py`: Handles CLI output formatting.
- `main.py`: Entry point of the application.

## How to Run
1. Open a terminal.
2. Navigate to the project directory.
3. Run the application:
   ```bash
   python main.py
   ```
